TapTalk Keyboard

Hold a button, speak, release. Your words are typed into whatever computer the board is plugged into, as a USB keyboard.

Before you install

  • A Waveshare ESP32-S3-Touch-AMOLED-1.8, revision V1. Turn it over: the sticker must say Display SH8601 and Touch FT3168. A V2 board (CO5300 / CST816) will flash successfully and then show a black screen.
  • Chrome or Edge, on a desktop. This page uses Web Serial, which Safari and Firefox do not implement.
  • A USB-C cable that carries data, not just power.
  • A Wi-Fi network and an OpenAI API key. You enter both on the device after flashing — nothing is baked into this firmware.
Your browser cannot talk to serial devices. Open this page in Chrome or Edge on a desktop. This page must be served over HTTPS.
If no device appears in the chooser, the board is not in download mode. It has no reset button — its EN pin is driven by the power-management chip. Hold BOOT, tap PWR, release BOOT, then click Install again.
If the board seems stuck after flashing, unplug and replug the USB-C cable. A USB-Serial-JTAG reset does not re-sample the boot strapping pins, so the chip can sit in download mode until it is power-cycled.

After it flashes

  1. The screen shows a QR code and an 8-digit password.
  2. Scan the code, or join the TapTalk-XXXX Wi-Fi from your phone.
  3. A setup page opens by itself. Enter your Wi-Fi and paste your API key.
  4. The device restarts and joins your network.

Tap Setup on the main screen to change or erase those later.

The API key is stored unencrypted. Anyone with this board and a USB cable can read it back. Use a project-scoped key with a spend limit, and erase it before lending the device. Why, and what it means.

What this build does and does not do

This is the bring-up build. It powers the display, touch, microphone, Wi-Fi and clock, and it shows a live microphone level bar. It does not yet transcribe speech or type on a USB keyboard — that lands next.

Deliberately, it installs no USB HID stack. Doing so would hand the USB port to the keyboard and take the serial console with it, which is also what makes this one-click installer work today.
