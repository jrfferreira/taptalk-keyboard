How it goes

  1. Flash itPlug the board in, click the button above, pick it from the chooser.

Getting an API key

  1. Open your OpenAI accountGo to platform.openai.com/api-keys and sign in. This is the developer platform, not ChatGPT — a ChatGPT Plus subscription does not include API access.
  2. Put a little credit on itUnder Billing, add a payment method and buy some credit. A key on an account with no balance returns insufficient_quota and the device shows an upload error. Five dollars goes a long way; transcription is billed per minute of audio (current rates).
  3. Create a project keyClick Create new secret key and scope it to a project used only by this device. If the key ever leaks, you revoke that one key and nothing else breaks.
  4. Set a spend limitUnder Settings → Limits, cap the monthly budget for that project. This is the single thing that turns a leaked key from a bill into an inconvenience.
  5. Copy it nowThe key starts with sk- and is shown exactly once. Paste it straight into the device's setup page — the easiest way is to open that page on the phone where your password manager lives.

Read this first

You need a V1 board

Waveshare sells two different boards under one name. Turn yours over: the sticker must say SH8601 and FT3168. A V2 board (CO5300 / CST816) flashes fine and then shows a black screen.

There is no reset button

The power chip drives the reset line. If no device shows up in the chooser, hold BOOT, tap PWR, release BOOT, and click Install again. If it seems stuck afterwards, unplug and replug.

Your API key is not encrypted

Anyone holding this board with a USB cable can read the key back out of flash. Use a project-scoped key with a spend limit, and erase it before you lend the device. The reasoning, in full.

Chrome or Edge, on a desktop

Installing uses the Web Serial API, which Safari and Firefox do not implement. Once flashed, the device needs no browser at all.
